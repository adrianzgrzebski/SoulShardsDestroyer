# SoulShardsDestroyer
An addon for Warlock (World of Warcraft) that destroy the excessive Soul Shards.<br />

# Commands for Addon: 
	/ssd 
		(display help info)
	/ssd set X
		(instead of X insert Your limit)
	/ssd check
		(check the limit)
